A Predictive Model to
Classify Performance of a Loan

In this project I attempt to model the performance of a loan, using a range of features such as borrower demoraphics, credit score, loan amount, number of credit lines, previous defaults, etc.
Beginning from a rather messy dataset, I would walk you through extensive data cleaning, feature engineering, feature selection, up-sampling, transformation pipeline building, hyper-parameter tuning, model selection, model training, testing and evaluating steps. Finally, I attempt to simplify the model by carefully examining the feature importances.
